# ProperDocs

## Environments

Check properdocs version.

```sh
uv run properdocs --version
```

```text
properdocs, version 1.6.7 from /workspaces/samples-tool/documentation/properdocs/.venv/lib/python3.14/site-packages/properdocs (Python 3.14)
```

## Create

Create documentation project.

```sh
uv run properdocs new samples
```

Run development server.

```sh
uv run properdocs serve
```

Build static documents.

```sh
uv run properdocs build
```
