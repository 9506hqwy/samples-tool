# Zensical

## Environments

Check zensical version.

```sh
uv run zensical --version
```

```text
0.0.28
```

## Create

Create documentation project.

```sh
uv run zensical new samples
```

Run development server.

```sh
uv run zensical serve
```

Build static documents.

```sh
uv run zensical build
```
